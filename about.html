About Us

Sign2Text is a research project made by 4 students at the University of Newcastle. Sign2Text can be used as a communication tool, or as an aid for learning American Sign Language.


What the buttons do

Test

Test displays the character with the highest probability score for the current video frame.

Translate

Translate is used to build sentences. Once a character has been read for 1.5 seconds consistently, it will add it to the input field.

Learn

The learn function is intended to help people practice their developing sign knowledge. The user will have 30 seconds to sign as many letters as they can. Each letter will appear on screen until the program reads a match. It will then randomly choose another letter to be displayed.
