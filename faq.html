Frequently Asked questions


Can I add my sign images to increase accuracy?

Yes, you can use the learn button to input pictures. For each alphabet, It will take 30 images.

How accurate is the application output?

On our test, It is xx% accurate.

How to add spaces in sentence?

3 second pause during the hand signs makes application to go for another word.

Why does sign2text use ASL (instead of something else / instead of AUSLAN etc)?

ASL is the most commonly used form of sign language in the world. Using ASL makes Sign2Text more accessible in general.

It isn't working / isn't accurate.

For best results, try using a solid background with good lighting and make sure your hand is contained withing the blue box.

The screen is black / can't see myself

Make sure your browser allows use of the webcam. If so, there should be a popup asking for permission when the page loads. Just click "allow".

Does it work on mobile / desktop?

Yes. Sign2Text will work both on mobile and desktop, provided it has access to a camera.

How does it work?

Sign2Text uses a convolutional neural network trained on thousands of hand images to make predictions of what gesture is being shown.

What about privacy /does it save images / can it access my data?

Sign2Text knows nothing about the person using it. Once the image has been processed to make a prediction, it is discarded and never used again. Nobody else has access to it.
